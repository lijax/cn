    python包管理工具pip安装方法

    包管理工具

    就像linux系统软件包的管理（debian系列的apt，redhat系列的yum）一样，几乎每种开源语言都有一个模块或包管理工具：

    语言 包管理工具 包主站
    python pip/easy_install Python Package Index
    perl cpan CPAN
    R install.packages CRAN
    Ruby gem RubyGems
    java maven Maven Central Repository

    python起初使用easy_install进行包的管理，pip对它进行了很多改进，功能上更强大，所以现在一般使用pip进行python包的管理。

    安装步骤

    安装环境：unbuntu

    • easy_install 包含在python-setuptools中，首先用系统包管理工具安装python-setuptools：

      sudo apt-get install python-setuptools

    • pip其实也是python的一个包，所以最后只需要用easy_install安装pip 即可：

      sudo easy_install pip
